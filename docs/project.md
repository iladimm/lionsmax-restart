# LionsMax Restart - Project Documentation

## Project Overview

**Project Name**: LionsMax Restart
**Target Audience**: Adults aged 40+
**Focus**: Mobile-first supplement e-commerce store
**Goal**: Provide high-quality health supplements with educational content

## Mission Statement

To empower adults 40+ to take control of their health through premium supplements and evidence-based nutritional guidance, delivered through a seamless mobile shopping experience.

## MVP Features

### Core Features
- [ ] Product catalog with filtering (by age group, health goal)
- [ ] Mobile-responsive design
- [ ] Product detail pages with ingredient information
- [ ] Shopping cart and checkout
- [ ] Blog/educational content section

### Phase 2 Features
- [ ] User accounts and order history
- [ ] Subscription service
- [ ] Personalized recommendations
- [ ] Email marketing integration
